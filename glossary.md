## Overview
Having a strong vocabulary available to you is one of the best tools that a language learner can have. This applies for French, Italian, English or even for the language of software development. On this page, I will record all of the terms I initially do not know, and attempt to record and explain them here succinctly.

## Terms
1. WSGI -- [Web Server Gateway Interface](https://www.google.com/search?client=firefox-b-1-d&q=WSGI) is an API and messaging standard between Python web applications and frameworks.
1. HashSet -- This structure holds a set of object (unordered, unique items) in a way that it allows you to quickly determine whether an object is in the set or not, usually within O(1). This differs from say, a List, in that scanning the list to determine if an object is contained runs in O(n) since the whole list has to be scanned, potentially. In Java this is a class which extends the Set class, and in Python it can be created using a dictionary.
1. [Window Sliding Technique](https://www.geeksforgeeks.org/window-sliding-technique/) -- The idea with this technique is if you have a list of items and you are searching for consecutive items which match a criteria, then you can place a 'window' over a certain set of those items and the 'slide' the window looking for matches. This reduces the computational complexity by eliminating nested `for` loops and potentially reducing an O(n^a) to O(n)
